What is a Table?

The <TABLE> tag was introduced for displaying content in a tabular format.

The main purpose of a table is the presentation of data.

Other uses would include data comparisons or viewing summarised data.

Anatomy of a Table

Basic elements of Table Tags
<table>...</table>to start and end the table
<tr> ... </tr>table row
<th> ... </th>table header
<td> ... </td>table data or cell definition
<caption> ... </caption>table caption (optional)

An HTML table consists of the <table> element and one or more <tr>, <th>, and <td> elements.

  • Tables are made from rows and columns.
  • The rows are defined with the <tr> tag as mentioned above.
  • The number of columns is determined by the number of <td> or <th> tags following each row command.
  • <th> cells are centered and bolded by default, while there is no default style applied to the table data (<td>).
  • The <caption> tag adds a caption above (before) the table.
<table>
 <caption>Example Table</caption>
 <tr>
    <th>heading 1</th>
    <th>heading 2</th>
    <th>heading 3</th>
 </tr>
 <tr>
    <td>data 1</td>
    <td>data 2</td>
    <td>data 3</td>
 </tr>
</table>

Adding a Border

The <table> element can have a border attribute that is supported in all major browsers. The border attribute specifies if a border should be displayed around the table cells or not.

You can leave out the border attribute and style the border in the .CSS file. You will need to add a border property for the <table> element, <th> element, and <td> element if you wish all elements to have a border as seen in the following example:

table, th, td {
  border: solid navy 2px;
}

Or if you only want (say) the <table> element to have a border only (and not the other elements), then you only need to add the border property for the table selector only.

table {
  border: solid navy 2px;
}
