Enhanced Multimedia Control

Among the new multimedia control features offered by Windows version 3.1 are:

  • Media Player, for control of
    • Sound cards
    • CD-ROM drives
    • Videodisc players
  • Sound Recorder, for
    • Sound files:
      • Playing sound files
      • Recording sound files
      • Editing sound files
    • Assigning sound files to application events
  • Control Panel enhancements include
    • Drivers option for easy driver set-up for virtually any device
    • Sound option for assigning unique sounds to system and application events
    • MIDI Mapper option for use with devices that have MIDI capabilities

For more information about these multimedia control features, refer to your Windows 3.1 manual.
