Objective

This lab looks at the many Bootstrap 4 Table classes.

  • Basic Table
  • Enhancing the Table
  • Table Head Styles
  • Contextual Classes
  • Responsive Tables

Basic Table

For a basic table with lightly padded cells and horizontal dividers, apply Bootstrap's .table class to the <table> element as in the following example.

<table class="table">
  ...
</table>

Inverse Table

You can invert the colours of the table by using the .table-dark class as in the following example.

<table class="table table-dark">
  ...
</table>

Enhancing the Table

Here, we will add stripes, a border and hover effect to the table. If the table takes up too much space, it can be condensed.

Striped Table

For alternating background colors on the rows, add Bootstrap's .table-striped class as in the following example.

<table class="table table-striped">
  ...
</table>

Bordered Table

For borders all around the table, use Bootstrap's table-bordered class as in the following example.

<table class="table table-bordered">
  ...
</table>

Hover Rows

To achieve that "hover" effect when mousing over the table rows, use Bootstrap's .table-hover class as in the following example.

<table class="table table-hover">
  ...
</table>

Small/Condensed Table

If you find a table has too much padding, you can half the padding by using Bootstrap's .table-sm class as in the following example.

<table class="table table-sm">
  ...
</table>

Table Head Styles

You can add a background color to the <thead> element by using either .thead-light or .thead-dark.

Default

Use the .thead-light class as in the following example.

<table class="table table-hover">
 <thead class="thead-light">
  <tr>
   <th>Header 1</th>
   <th>Header 2</th>
   <th>Header 3</th>
  </tr>
 </thead>
  ...
</table>

Inverse

Use the .thead-dark class as in the following example.

<table class="table table-hover">
 <thead class="thead-dark">
  <tr>
   <th>Header 1</th>
   <th>Header 2</th>
   <th>Header 3</th>
  </tr>
 </thead>
  ...
</table>

Contextual Classes

You can apply color to individual rows or cells by using Bootstrap's contextual classes as seen in the following example.

The 5 contextual classes are:

  • .table-active,
  • .table-danger,
  • .table-info,
  • .table-success, and
  • .table-warning.

Responsive Tables

Responsive tables allow tables to be scrolled horizontally with ease. Make any table responsive across all viewports by adding .table-responsive class on .table. Or, pick a maximum breakpoint with which to have a responsive table up to by adding .table-responsive{-sm|-md|-lg|-xl}..
