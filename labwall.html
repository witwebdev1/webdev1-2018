Website Development 1

All labs's in Module

    Assignment One

    This assignment is worth 30% of your grade for this module.

    Assignment Two

    This assignment is worth 55% of your grade for this module.

    Create your first HTML page

    This lab will firstly introduce you to the tools we will use during the web development module and secondly introduce you to creating, editing, saving and displaying a web page.

    Formatting text

    This lab will introduce you to the HTML elements that allow you to format how text appears on your web pages.

    Adding an image

    This lab will explain how to add an image to your web page. We'll do much more with images in the coming weeks but, for now, the basics.

    Adding lists

    This lab will show you how to add numbered and bulleted lists to your web page.

    Linking pages

    In this lab, you will learn how web pages are linked together.

    Validating HTML

    In this lab, you will use an online tool to validate your HTML code and check that it is correctly structured.

    CSS Introduction

    This lab will introduce you to the concepts of Cascading Style Sheets

    Adding Style

    This lab will introduce you to the methods of adding CSS style rules to a web page

    Adding Colour

    This lab will introduce you to the methods of adding colour.

    Controlling the Font

    This lab explains how to set the Font properties.

    Text Properties

    This lab explains how to set the Text properties.

    Styling Links

    This lab will show you how to style links and the different link states.

    Grouping Styles

    This lab will show you how to group styles together.

    Id and Class Selectors

    This lab looks at Class and Id selectors.

    Descendant Selectors

    This lab looks at descendant selectors.

    Styling Lists

    This lab explores how to style lists.

    CSS Part Two Lab

    The purpose of this lab is to get practice using CSS selectors, classes and id selectors; descendant selectors; and styling links.

    Table Introduction

    This lab introduces what a table is and describes its anatomy.

    Table Styling

    This lab looks at styling tables.

    Advanced Tables

    This lab looks at merging table cells.

    Using nth-of-type

    This lab demonstrates how to select specific table rows by using the nth-of-type selector.

    HTML Tables Lab

    The purpose of this practical session is to get comfortable with adding and styling tables; and selecting and styling different portions of a table.

    Set up 000 web host

    In this lab you will create an account with 000webhost, a web hosting company that provide free accounts for learners. You will also deploy a website that we have worked on over the past few weeks.

    Set up 5gbfree

    In this lab you will create an account with 5gfree, a web hosting company that provide free accounts for learners. You will also deploy a website that we have worked on over the past few weeks.

    The Box Model

    In this lab, we introduce the box model and examine related properties: padding, margin, border, outline, width, height and overflow. We also look at the <div> and <span> elements in more detail.

    Layout and Positioning

    In this lab, you will learn more about the CSS properties used for layout, including positioning, floating elements, and how to handle overlapping elements.

    Background Images

    In this lab, you will learn how to add background images to your webpages and control their appearance. You will also apply a gradient background to a webpage.

    Semantic Layout

    This lab introduces the use of semantic structural elements that can be used for page layout.

    Figures and captions

    In this lab, you will use the figure and figcaption tags to style images and their captions.

    Navigation

    In this lab, you will learn how to create a horizontal navigation menu. Complete the first exercise detailed in steps 1, 2 and 3 (WIT website); then, complete the additional exercise at the end (Tresidder Union).

    Semantic UI Grid

    This lab looks at Semantic UI Containers, and the Semantic UI Grid System.

    Semantic UI Elements

    This lab looks at some Semantic UI elements.
